# Modules

| File | Purpose |
|---|---|
| `games.py` | Zero-sum payoff matrices (RPS, matching pennies, random) |
| `opponents.py` | Five opponent types for the benchmark suite |
| `solver.py` | `AdaptiveSolver`: MWU + drift detection |
| `baseline.py` | Three baselines: Nash maximin, fixed Hedge, Bayesian best-response |
| `experiments.py` | Benchmark runner, regret computation, speedup metrics |
| `plot_results.py` | Convergence curves and speedup bar chart |
